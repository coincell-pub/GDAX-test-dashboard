A test purpose GDAX price dashboard

## Requirements
Node.js
Screen >= 22"(recommended)

## Installation
```
git clone https://github.com/coincell-pub/GDAX-test-dashboard
cd GDAX-test-dashboard
npm install
```

## Config
Edit `config.ini` in the application's directory.

## Running
`./cryptocharts.js`
